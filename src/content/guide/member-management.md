# अध्याय 5 — सदस्य व सदस्यता प्रबंधन

> **इस अध्याय के बाद आप:** सदस्य जोड़ना व अनुमोदन; शेयर पूँजी व प्रवेश शुल्क की अलग प्रविष्टि; सदस्य-बही; निकासी पर शेयर-वापसी।

## 5.1 सदस्य ही धुरी
"एक सदस्य, एक वोट।" पूँजी का मुख्य स्रोत सदस्यों की शेयर पूँजी; लाभ का हिस्सा लाभांश व संरक्षण-छूट के रूप में लौटता है।

## 5.2 सदस्यता के प्रकार
नियमित (शेयर पूँजी, पूर्ण अधिकार) व नाममात्र (सीमित अधिकार)।

## 5.3 नया सदस्य जोड़ना
Members → Add Member → नाम, जुड़ने की तिथि, शेयर संख्या व शेयर पूँजी → सहेजें (सदस्य-ID व खाता स्वतः)।

> 💡 जुड़ने की तिथि सही भरें — Federation रिपोर्ट का आधार।

## 5.4 आवेदन व अनुमोदन
Member Application → लंबित आवेदन → Approve/Reject। केवल अनुमोदित सदस्य रिपोर्ट/लाभांश में गिने जाते हैं।

## 5.5 शेयर पूँजी की प्रविष्टि
राम ने 10 शेयर @ ₹100 = ₹1,000 नकद:

| खाता | Dr | Cr |
|---|---|---|
| नकद | 1,000 | |
| व्यक्तिगत शेयर पूँजी (1102) | | 1,000 |

> ⚠️ शेयर पूँजी को **आय न मानें** — यह पूँजी है।

## 5.6 प्रवेश शुल्क (यह आय है)
₹1,000 शेयर पूँजी + ₹50 प्रवेश शुल्क, कुल ₹1,050:

| खाता | Dr | Cr |
|---|---|---|
| नकद | 1,050 | |
| शेयर पूँजी (1102) | | 1,000 |
| प्रवेश शुल्क (आय) | | 50 |

## 5.7 सदस्य-बही
हर सदस्य की अलग बही। सभी सदस्यों की शेयर पूँजी का योग = खाता 1102 शेष = तुलन-पत्र/रिटर्न।

## 5.8 निकासी पर शेयर-वापसी
Dr शेयर पूँजी (1102) / Cr नकद-बैंक। सदस्य को **निष्क्रिय** करें, हटाएँ नहीं।

## ⚠️ आम गलतियाँ

| गलती | बचाव |
|---|---|
| शेयर पूँजी को आय | 1102 (पूँजी) |
| प्रवेश शुल्क को पूँजी | आय |
| बिना अनुमोदन गिनना | पहले Approve |
| सदस्य हटाना | निष्क्रिय करें |

## 🔍 ऑडिट टिप्पणी
"सदस्य रजिस्टर अद्यतन; शेयर पूँजी कुल 1102 से मिलान; प्रवेश शुल्क आय में वर्गीकृत।"

## 📘 सहकारी मानक
प्रत्येक सदस्य को न्यूनतम एक शेयर अनिवार्य; शेयर पूँजी = स्वामित्व-पूँजी, ऋण नहीं।

## ✅ हल किया हुआ उदाहरण
तीन नए सदस्य, प्रत्येक 10 शेयर @ ₹100 + ₹50 प्रवेश शुल्क नकद:

| खाता | Dr | Cr |
|---|---|---|
| नकद | 3,150 | |
| शेयर पूँजी (1102) | | 3,000 |
| प्रवेश शुल्क (आय) | | 150 |

## 📚 केस-स्टडी
प्रवेश शुल्क को शेयर पूँजी में जोड़ती रही → पूँजी बढ़ी-चढ़ी, आय कम। सुधार के बाद प्रवेश शुल्क अलग आय में।

## ❓ FAQ
**प्र. सदस्य और शेयर ले?** नई प्रविष्टि (Dr नकद/बैंक, Cr 1102)।
**प्र. कुल शेयर पूँजी कहाँ?** Members पेज = खाता 1102 शेष।
