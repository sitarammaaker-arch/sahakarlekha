# परिशिष्ट B — शब्दावली (Glossary)

|---|---|

| क्रेडिट (Cr) | खाते का दायाँ पक्ष |
| वाउचर | लेन-देन की एंट्री/प्रमाण |

| आमदनी-खर्च खाता | नेट प्रॉफ़िट/घाटा |
| बैलेंस शीट | स्थिति-विवरण (संपत्ति=देनदारी+पूँजी) |
| रसीद-भुगतान | नकदी-आधार सारांश |
| उपार्जन-आधार | घटना-आधारित (नकदी नहीं) |

| बँटवारा | लाभ का बँटवारा |
| रिज़र्व फंड | अनिवार्य संचय (≥25%) |
| व्यवहार बोनस | सदस्य-लेन-देन पर वापसी |
| वेटेड एवरेज लागत | औसत खरीद-लागत |
| FIFO | पहले आया पहले गया |
| डेप्रिसिएशन | संपत्ति-घिसाव-व्यय |
| WDV | घटते-शेष मूल्य |
| ITC | इनपुट टैक्स क्रेडिट |
| TDS | स्रोत-कटौती |
| NPA | अनर्जक आस्ति (90+ दिन) |
| RLS | पंक्ति-स्तरीय सुरक्षा |
| Maker-Checker | निर्माता-जाँचकर्ता नियंत्रण |
