# Appendix I — Troubleshooting & Tricky Cases

## I.1 Problem → fix

| Problem | Cause | Fix |
|---|---|---|
| Trial balance won't tally | one-sided/wrong amount | check ÷2, ÷9 |
| Cash negative | receipt missed/double payment | reconcile cash daily |
| Bank mismatch | charges/interest/uncleared | BRS; missing entry |
| Stock mismatch | update missed | physical count; Shortage/Excess |
| Data gone on F5 | cloud-save failed | re-enter on the red warning |
| HAFED mismatch | claim/recovery unrecorded | reconcile the agency account |
| Gunny-bag shortage | register not updated | received = used + returned + balance |

## I.2 Tricky cases (decision)
- Looks like an expense but is an **asset:** security/guarantee deposit, advances, advance tax.
- Looks like income but is a **liability:** deposit, customer advance, GST/TDS collected.
- Looks like a purchase but is an **asset:** durable own-use item (furniture/vehicle).
- **Recovery vs sale:** a credit sale is recorded first; later money is only recovery.
