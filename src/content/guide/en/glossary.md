# Appendix B — Glossary

| Term | Meaning |
|---|---|
| Debit (Dr) | The left side of an account |
| Credit (Cr) | The right side of an account |
| Voucher | The entry/proof of a transaction |
| Ledger | An account-wise page of transactions |
| Day Book | A time-ordered list of vouchers |
| Trial Balance | A list of all balances, Dr = Cr |
| Trading Account | Gross profit/loss |
| Income & Expenditure | Net surplus/deficit |
| Balance Sheet | The position statement (assets = liabilities + capital) |
| Receipts & Payments | A cash-basis summary |
| Accrual basis | Event-based (not cash) |
| Surplus | A cooperative’s "profit" |
| Appropriation | The sharing of the surplus |
| Statutory Reserve | A mandatory reserve (≥25%) |
| Patronage Rebate | A return on members’ transactions |
| Weighted-average cost | The average purchase cost |
| FIFO | First in, first out |
| Depreciation | The asset wear-and-tear expense |
| WDV | Written-down value |
| ITC | Input Tax Credit |
| TDS | Tax Deducted at Source |
| NPA | Non-Performing Asset (90+ days) |
| RLS | Row-Level Security |
| Maker-Checker | Maker–checker control |
| FY-Lock | Audit-locking of the financial year |
| Bill-wise settlement | Linking a payment to a bill |
| Ageing | Age-classification of dues |
