# अध्याय 11 — वेतन प्रबंधन (Salary)

> **इस अध्याय के बाद आप:** कर्मचारी मास्टर व slip; सकल/कटौती/शुद्ध; वेतन की प्रविष्टि; EPF/ESI/PT/TDS जमा।

## 11.1 वेतन क्यों ख़ास
सकल − कटौतियाँ = शुद्ध। कटौतियाँ = समिति की देयता (सरकार को देय)।

## 11.2 कर्मचारी मास्टर
Salary → कर्मचारी जोड़ें: नाम, पद, सकल वेतन, EPF/ESI/PT लागू?

## 11.3 घटक

| घटक | उदाहरण (₹) |
|---|---|
| सकल | 20,000 |
| − EPF | 1,800 |
| − ESI | 150 |
| − PT | 200 |
| − TDS | 500 |
| = शुद्ध | 17,350 |

## 11.4 slip बनाना
कर्मचारी + माह → कटौतियाँ → शुद्ध → slip बनाएँ → PDF।

## 11.5 प्रविष्टि

| खाता | Dr | Cr |
|---|---|---|
| वेतन (व्यय) | 20,000 | |
| देय EPF (2203) | | 1,800 |
| देय ESI (2204) | | 150 |
| देय PT (2207) | | 200 |
| देय TDS (2202) | | 500 |
| देय वेतन/नकद | | 17,350 |

> 💡 पूरा सकल व्यय है; कर्मचारी को शुद्ध मिलता है।

## 11.6 कटौतियाँ जमा
Dr देय EPF/ESI/PT/TDS / Cr बैंक।

## 11.7 वेतन रजिस्टर
माह-वार slip, कटौतियाँ, शुद्ध; "देय EPF" = खाता 2203 शेष।

## 11.8 बोनस/ग्रेच्युटी
नियमित बोनस=व्यय(5207); लाभ-में-से बोनस=**विनियोग/देय**; ग्रेच्युटी=प्रावधान(5206)।

> ⚠️ लाभ-बोनस को व्यय 5207 में न डालें — देय में।

## ⚠️ आम गलतियाँ

| गलती | बचाव |
|---|---|
| शुद्ध को व्यय मानना | पूरा सकल व्यय |
| कटौतियाँ जमा न करना | 2202–2207 जाँचें |
| लाभ-बोनस को व्यय में | देय में |

## 🔍 ऑडिट टिप्पणी
"वैधानिक कटौतियाँ काटी व समय पर जमा; देय खाते सत्यापित।"

## 📘 श्रम मानक
EPF/ESI/TDS कटौती व नियत तिथि तक जमा वैधानिक दायित्व।

## ✅ हल किया हुआ उदाहरण
सकल ₹15,000; EPF ₹1,200, PT ₹150 → शुद्ध ₹13,650; Dr वेतन 15,000 / Cr देय EPF 1,200 / Cr देय PT 150 / Cr नकद 13,650।

## 📚 केस-स्टडी
केवल शुद्ध को व्यय मानती रही → कटौती-देय न दिखे, समय पर जमा न हुई। सुधार के बाद सही।

## ❓ FAQ
**प्र. EPF/ESI लागू नहीं?** शून्य रखें; सकल=शुद्ध।
**प्र. नियोक्ता अंश?** अलग व्यय-प्रविष्टि।

## ✏️ अभ्यास
1. सकल ₹18,000; EPF ₹1,500, TDS ₹300 — शुद्ध व प्रविष्टि।
2. देय TDS ₹300 बैंक से जमा।
3. सही/गलत: "केवल शुद्ध व्यय होता है।"
4. लाभ-बोनस व्यय या विनियोग?
5. "देय EPF" मिलान किस खाते से?
