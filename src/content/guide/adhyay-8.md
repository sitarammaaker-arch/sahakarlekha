# अध्याय 8 — खरीद प्रबंधन (Purchases)

> **इस अध्याय के बाद आप:** नकद/उधार खरीद; GST इनपुट क्रेडिट; TDS कटौती (एक बार); क्रय रजिस्टर।

## 8.1 तीन असर
खरीद से: स्टॉक बढ़ता, लागत दर्ज, नकद जाता/आपूर्तिकर्ता को देय।

## 8.2 नकद/उधार खरीद
नकद: Dr उर्वरक खरीद (5110) 45,000 / Cr नकद 45,000 (स्टॉक +100)।
उधार: Dr उर्वरक खरीद 45,000 / Cr इफको (2101) 45,000।

## 8.3 GST इनपुट क्रेडिट (संपत्ति)
₹10,000 + 18% GST, उधार:

| खाता | Dr | Cr |
|---|---|---|
| उपभोक्ता खरीद (5112) | 10,000 | |
| GST इनपुट क्रेडिट (3310) | 1,800 | |
| आपूर्तिकर्ता (2101) | | 11,800 |

> ⚠️ GST को खरीद-लागत में न जोड़ें।

## 8.4 TDS — दो बार न घटाएँ
grandTotal = कर-योग्य + GST − TDS। देय = grandTotal; कटा TDS अलग 2202 में।
₹1,00,000, TDS 0.1% = ₹100:

| खाता | Dr | Cr |
|---|---|---|
| खरीद | 1,00,000 | |
| आपूर्तिकर्ता | | 99,900 |
| देय TDS (2202) | | 100 |

## 8.5 ऐप में
Purchases → तिथि, आपूर्तिकर्ता, वस्तुएँ → भुगतान माध्यम → सहेजें (स्टॉक बढ़ता)।

## 8.6 प्रति-वस्तु खरीद खाता व स्टॉक
गतिविधि-खाते (5110…); खरीद से स्टॉक व लागत-दर अपडेट।

## 8.7 क्रय रजिस्टर
तिथि-वार खरीद, GST, TDS, स्थिति। GSTR-3B/TDS/ऑडिट हेतु निर्यात।

## 8.8 संशोधन/विलोपन
Purchase Management से — स्टॉक घटता, वाउचर रद्द, आपूर्तिकर्ता-बही सही।

## ⚠️ आम गलतियाँ

| गलती | बचाव |
|---|---|
| GST को लागत में | 3310 (ITC) |
| TDS दो बार | देय = grandTotal |
| आपूर्तिकर्ता न चुनना | अवश्य |

## 🔍 ऑडिट टिप्पणी
"इनपुट GST पृथक्; TDS कटौती व जमा सही; स्टॉक-आगम खरीद से मिलान।"

## 📘 सहकारी मानक
खरीद उपार्जन आधार पर; ITC व TDS का सही अभिलेख।

## ✅ हल किए हुए उदाहरण
नकद: Dr बीज खरीद 1,750 / Cr नकद 1,750। उधार+GST: Dr कीटनाशक 20,000 / Dr ITC 3,600 / Cr आपूर्तिकर्ता 23,600। TDS: Dr खरीद 2,00,000 / Cr आपूर्तिकर्ता 1,99,800 / Cr देय TDS 200।

## 📚 केस-स्टडी
TDS दो बार घटाकर वाउचर असंतुलित। देय = grandTotal करते ही संतुलित।

## ❓ FAQ
**प्र. ITC कब वापस?** बिक्री-GST से समायोजित।
**प्र. GST नहीं?** फ़ील्ड शून्य; सीधी प्रविष्टि।

## ✏️ अभ्यास
1. ₹8,000 उपभोक्ता नकद (बिना GST) — प्रविष्टि।
2. ₹10,000 + 18% GST उधार खरीद।
3. ₹1,50,000, TDS 0.1%, उधार — देय व प्रविष्टि।
4. ITC किस खाते में, संपत्ति या व्यय?
5. खरीद हटाने पर तीन चीज़ें।
