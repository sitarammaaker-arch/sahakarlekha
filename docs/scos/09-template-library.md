## Governance for templates
1. Every `⚠️` template **must be validated** (statutory formats differ by state/year) before gating it as a magnet.
2. Each template's landing page links to the **module that produces it automatically** → conversion bridge.
3. Register new downloads in a single **download manifest** (powers `/downloads` hub + sitemap + lead `source`).
4. English content for jsPDF magnets; Excel/Word may carry Hindi labels.
5. Version + "last validated" stamp on each file.
