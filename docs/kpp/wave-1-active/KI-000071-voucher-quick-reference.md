# KI-000071 · Voucher Entry Quick Reference (वाउचर एंट्री त्वरित संदर्भ)

**Definition:** A voucher entry quick reference is a short "which voucher for what" guide — it maps common situations (money received, money paid, cash↔bank, adjustments) to the right voucher type, so users pick correctly without guessing.

**Plain-language explanation:** It is a cheat sheet: money in → receipt, money out → payment, cash↔bank → contra, adjustments → journal. Pick the right voucher fast.

**Hindi explanation (आसान हिंदी):** वाउचर एंट्री त्वरित संदर्भ एक छोटी "किस काम के लिए कौन-सा वाउचर" गाइड है — पैसा मिला → रसीद (receipt), पैसा गया → भुगतान (payment), कैश↔बैंक → contra, समायोजन → journal। इससे उपयोगकर्ता बिना अटके सही वाउचर चुन लेता है।

**English explanation:** A quick map from everyday situations to the correct voucher type — the fastest way for new staff to enter transactions correctly.

**Why it matters:** Choosing the wrong voucher type is a common beginner mistake; a quick reference prevents it and keeps the books clean.

**Common misconceptions:**
- ❌ "Any voucher type works for any entry." → The right type keeps reports and books correct.
- ❌ "You must memorise everything first." → A quick-reference card lets you start immediately.
