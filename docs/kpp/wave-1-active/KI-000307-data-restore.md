# KI-000307 · Data Restore (डेटा बहाली)

**Definition:** Data restore is the process of bringing back the society's data from a backup — used to recover after accidental loss, damage, or a wrong change, returning the records to a known-good state.

**Plain-language explanation:** Restore is using a backup to get your data back if something goes wrong — the other half of "backup".

**Hindi explanation (आसान हिंदी):** डेटा रिस्टोर (बहाली) वह प्रक्रिया है जिसमें बैकअप से समिति का डेटा वापस लाया जाता है — गलती से डेटा खोने, खराब होने या गलत बदलाव के बाद रिकॉर्ड को पिछली सही स्थिति में लौटाने के लिए। यह "बैकअप" का दूसरा आधा हिस्सा है।

**English explanation:** Recovering data from a backup to a known-good state; backup without a working restore is incomplete.

**Why it matters:** A backup is only useful if it can be restored. Understanding restore gives confidence that the society's records are truly safe.

**Common misconceptions:**
- ❌ "Backups alone keep me safe." → You also need a working restore to actually recover.
- ❌ "Restoring is risky and complicated." → In good software it is a guided, safe step.
