# KI-000097 · Account-Name Glossary (खातों के नामों की शब्दावली)

**Definition:** An account-name glossary is a standard list of ledger account names in Hindi and English — so a society uses consistent, recognised names (e.g. "किराया / Rent", "ब्याज आय / Interest Income") instead of ad-hoc ones.

**Plain-language explanation:** It is a ready list of proper account names in both languages, so everyone names accounts the same way.

**Hindi explanation (आसान हिंदी):** खातों के नामों की शब्दावली एक मानक सूची है जिसमें लेजर खातों के नाम हिंदी व अंग्रेज़ी दोनों में दिए होते हैं — ताकि समिति मनमाने नामों की जगह एक जैसे, पहचाने हुए नाम इस्तेमाल करे (जैसे "किराया / Rent", "ब्याज आय / Interest Income")।

**English explanation:** A bilingual reference of standard ledger-account names, keeping the chart of accounts consistent across users and societies.

**Why it matters:** Consistent names make reports comparable and audits smoother, and help new staff set up accounts correctly.

**Common misconceptions:**
- ❌ "Account names don't matter." → Consistent names keep reports clear and comparable.
- ❌ "Every society must invent its own names." → A standard glossary saves time and avoids errors.
