# KI-000303 · Cloud Accounting (क्लाउड लेखांकन)

**Definition:** Cloud accounting is keeping a society's books in secure online software accessed through a browser, where data is stored on remote servers and is available from anywhere, instead of on one local computer or paper registers.

**Plain-language explanation:** It means doing the society's accounts on a website — your data lives safely online, so you can use it from any device and never lose a paper register.

**Hindi explanation (आसान हिंदी):** क्लाउड लेखांकन यानी समिति का हिसाब किसी सुरक्षित ऑनलाइन सॉफ़्टवेयर में रखना, जिसे ब्राउज़र से चलाया जाता है। डेटा ऑनलाइन सर्वर पर सुरक्षित रहता है और कहीं से भी इस्तेमाल हो सकता है — न एक ही कंप्यूटर की मजबूरी, न कागज़ी रजिस्टर खोने का डर। SahakarLekha इसी तरह काम करता है।

**English explanation:** Browser-based accounting with data stored on secure remote servers, accessible anywhere — the model SahakarLekha uses.

**Why it matters:** It explains how SahakarLekha works and why it is safer and more flexible than local files or paper — a core selling point for societies going digital.

**Common misconceptions:**
- ❌ "Cloud means my data is less safe." → Reputable cloud systems keep encrypted, backed-up copies — usually safer than a single local PC.
- ❌ "I need to install heavy software." → A browser is enough; there is nothing big to install.
