# Wave-1 Activation — Index (50 Knowledge Items)

> The first **activated** Level-A Knowledge Items. Each file is a fully-populated, independent,
> reusable knowledge record (definition, Hindi/English explanations, why-it-matters, misconceptions,
> relationships, internal links, suggested assets, search intent, persona, learning objectives).
>
> **Scope:** Level-A only (educational / product). **No** accounting treatments, legal advice, tax
> rates, or jurisdiction-specific compliance — those remain `planned` pending SME (E3).
> **Status:** these 50 are `active` in the [registry](../05-wave-1-registry.md); the other 306 Wave-1
> KIs stay `planned`.

**Each KI obeys:** one concept = one KI · references SCOS (`C###`) + SMRD (`SMRD:C###`) + KAE (`EV-`) ·
beginner-friendly Hinglish · single source of truth (no duplication). These are **records, not articles** —
articles/FAQs/videos are *generated from* them later ([KI→assets map](../00-master-index.md)).
