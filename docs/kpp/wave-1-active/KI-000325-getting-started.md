# KI-000325 · Getting Started (शुरुआत कैसे करें)

**Definition:** Getting started is the recommended first-path for a new society on SahakarLekha — register, set up the society, add ledger heads/members, enter opening balances, and record the first voucher — to reach a working set of books quickly.

**Plain-language explanation:** It is the simple starting checklist: sign up, set up the society, add your accounts and members, put in opening balances, and make your first entry.

**Hindi explanation (आसान हिंदी):** "शुरुआत कैसे करें" नए उपयोगकर्ता के लिए सुझाया गया पहला रास्ता है — रजिस्टर करें, समिति सेटअप करें, ज़रूरी खाते (ledger heads) व सदस्य जोड़ें, ओपनिंग बैलेंस (opening balance) डालें, और पहला वाउचर बनाएँ। इससे जल्दी ही काम-लायक बहीखाता तैयार हो जाता है। आसान शब्दों में — "कहाँ से और किस क्रम में शुरू करें"।

**English explanation:** The onboarding sequence that takes a new user from sign-up to a functioning set of books in a few logical steps.

**Why it matters:** New users often don't know where to begin. A clear getting-started path reduces drop-off and gets a society productive fast.

**Common misconceptions:**
- ❌ "I must learn everything before starting." → The first few steps are enough to begin; the rest can be learned as you go.
- ❌ "I should enter this year's transactions before setting up." → Setup and opening balances come first, then daily vouchers.
