# KI-000022 · MPACS — Multi-Purpose PACS (बहुउद्देशीय PACS)

**Definition:** An MPACS is a multi-purpose primary agricultural credit society that, beyond crop credit, runs several member services together — such as input supply, PDS/ration distribution, storage, and procurement — at the village level.

**Plain-language explanation:** It is a PACS that does many things at once — not just loans, but also supplying inputs, running a ration shop, storage, and more for the village.

**Hindi explanation (आसान हिंदी):** MPACS यानी बहुउद्देशीय प्राथमिक कृषि साख समिति — वह PACS जो सिर्फ़ फसली कर्ज़ ही नहीं, बल्कि कई सेवाएँ एक साथ देती है — जैसे खाद-बीज की आपूर्ति, राशन (PDS) वितरण, भंडारण और खरीद — गाँव स्तर पर।

**English explanation:** A village PACS expanded into multiple activities (credit + supply + PDS + storage), serving as a one-stop cooperative for farmer-members.

**Why it matters:** MPACS need multi-activity accounting (loans, stock, supply); SahakarLekha can keep these segregated and consolidated.

**Common misconceptions:**
- ❌ "MPACS and PACS are unrelated." → An MPACS is a PACS doing multiple activities.
- ❌ "One combined account covers everything." → Each activity should be tracked clearly for audit.
