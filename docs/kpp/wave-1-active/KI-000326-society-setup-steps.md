# KI-000326 · Society Setup Steps (समिति सेटअप के चरण)

**Definition:** Society setup steps are the ordered first-time tasks to configure a society in the software — enter the society's name and type, set the financial year, confirm the chart of accounts, and add basic details — so daily entry can begin correctly.

**Plain-language explanation:** It is the short checklist to set up your society the first time: name, type, financial year, accounts — then you're ready to start entering.

**Hindi explanation (आसान हिंदी):** समिति सेटअप के चरण वे क्रमबद्ध पहली-बार के काम हैं जिनसे सॉफ्टवेयर में समिति तैयार होती है — समिति का नाम व प्रकार डालें, वित्तीय वर्ष सेट करें, चार्ट ऑफ़ अकाउंट्स पक्का करें, और बुनियादी जानकारी जोड़ें। इसके बाद रोज़ की एंट्री सही ढंग से शुरू हो सकती है।

**English explanation:** The step-by-step first-run configuration; doing it in order (especially type + financial year) makes all later reports work correctly.

**Why it matters:** A clean setup is the foundation — type and financial year affect every later report. Doing the steps in order avoids rework.

**Common misconceptions:**
- ❌ "I can skip setup and start entering." → Core setup (type, FY, accounts) must come first.
- ❌ "Setup is repeated each year." → It is a one-time foundation; only year-end/opening repeats.
