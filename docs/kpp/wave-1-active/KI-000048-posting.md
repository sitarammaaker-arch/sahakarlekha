# KI-000048 · Posting (पोस्टिंग)

**Definition:** Posting is the act of transferring entries from the journal/voucher into the relevant ledger accounts, so that each account gathers all its transactions and shows its balance.

**Plain-language explanation:** Posting means copying each entry into the right account's page, so every account's running total stays up to date.

**Hindi explanation (आसान हिंदी):** पोस्टिंग का मतलब है — जर्नल/वाउचर की एंट्री को संबंधित खाता बही (ledger) के खातों में चढ़ाना, ताकि हर खाते में उसके सारे लेन-देन इकट्ठा हों और शेष (balance) सही दिखे। आसान शब्दों में — एंट्री को सही "खाते के पन्ने" पर ले जाना।

**English explanation:** Moving a recorded entry into its ledger account(s); in software this happens automatically the moment a voucher is saved.

**Why it matters:** Posting is what turns scattered entries into account-wise balances that build the trial balance and reports. SahakarLekha posts automatically.

**Common misconceptions:**
- ❌ "Posting is a separate manual step in software." → Modern software posts vouchers to ledgers instantly.
- ❌ "Posting changes the amount." → It only moves the same entry into the account; amounts don't change.
