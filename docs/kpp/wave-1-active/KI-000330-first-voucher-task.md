# KI-000330 · Make the First Voucher (पहला वाउचर कैसे करें)

**Definition:** Making the first voucher is the task of entering a society's first transaction in the software — choosing the right voucher type (receipt/payment/journal/contra), the accounts, amount, date, and narration — to confirm the books are working.

**Plain-language explanation:** It is recording your first transaction: pick the voucher type (money in/out etc.), choose accounts, enter the amount and a note, and save.

**Hindi explanation (आसान हिंदी):** पहला वाउचर करना यानी सॉफ्टवेयर में समिति का पहला लेन-देन दर्ज करना — सही वाउचर प्रकार (रसीद/भुगतान/journal/contra) चुनना, खाते, रकम, तारीख और विवरण डालना, और सेव करना। इससे पक्का होता है कि बही ठीक चल रही है।

**English explanation:** The step-by-step task of recording the first transaction; doing one correctly builds confidence and confirms the setup works.

**Why it matters:** The first voucher is the moment a society's books "go live". Doing it right sets the pattern for all daily entry.

**Common misconceptions:**
- ❌ "Any voucher type will do for the first entry." → Choose the type that matches the transaction.
- ❌ "Narration can be skipped on the first entry." → A clear narration is good practice from the start.
