# KI-000305 · Society Setup (समिति सेटअप)

**Definition:** Society setup is the one-time configuration done when a society starts using SahakarLekha — entering basic details like name, society type, financial year, and initial chart of accounts — so the system is ready for daily entries.

**Plain-language explanation:** It is the first-time setup: tell the software who the society is, its type, and its year, so everything afterwards works correctly.

**Hindi explanation (आसान हिंदी):** समिति सेटअप वह एक बार किया जाने वाला कॉन्फ़िगरेशन है जो SahakarLekha शुरू करते समय किया जाता है — समिति का नाम, प्रकार, वित्तीय वर्ष और शुरुआती चार्ट ऑफ़ अकाउंट्स डालना। इससे सिस्टम रोज़ के काम के लिए तैयार हो जाता है। आसान शब्दों में — "पहली बार की ज़रूरी सेटिंग"।

**English explanation:** The initial configuration step that prepares the software for a specific society before any vouchers are entered.

**Why it matters:** Correct setup (especially society type and financial year) makes every later report and feature work right. It is the recommended first task for new users.

**Common misconceptions:**
- ❌ "Setup must be perfect before I can start." → Core details matter most; many settings can be refined later.
- ❌ "Setup is repeated every year." → It is a one-time foundation; only year-end/opening steps repeat.
