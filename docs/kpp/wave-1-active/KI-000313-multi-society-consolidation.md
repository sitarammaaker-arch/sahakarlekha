# KI-000313 · Multi-Society Consolidation (बहु-समिति समेकन)

**Definition:** Multi-society consolidation is the ability to manage and view several societies together in one place — useful for a federation, an apex body, or an accountant/auditor serving many societies — with each society's books kept separate but viewable side by side.

**Plain-language explanation:** It lets one login handle many societies, each with its own books, while still seeing a combined picture — handy for federations and shared accountants.

**Hindi explanation (आसान हिंदी):** बहु-समिति समेकन यानी कई समितियों को एक ही जगह से संभालना व देखना — संघ, शीर्ष संस्था, या कई समितियों को संभालने वाले लेखाकार/ऑडिटर के लिए उपयोगी। हर समिति की बही अलग रहती है, पर उन्हें साथ-साथ देखा जा सकता है।

**English explanation:** Managing multiple societies from one place with separate books but a combined view — for federations and multi-society professionals.

**Why it matters:** Federations and shared accountants need oversight across societies without mixing their books. SahakarLekha supports this.

**Common misconceptions:**
- ❌ "Consolidation merges everyone's books into one." → Each society's books stay separate; only the view is combined.
- ❌ "It's only for big federations." → Any accountant handling several societies benefits.
